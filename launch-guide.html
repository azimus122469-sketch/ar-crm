1
Создать аккаунт на GitHub
Бесплатно · 3 минуты
1
Открой сайт
Зайди на github.com с компьютера или телефона.
github.com
2
Нажми «Sign up»
Кнопка в правом верхнем углу. Введи email, пароль, придумай имя пользователя.
⚠️
Запомни имя пользователя — оно войдёт в адрес твоего приложения. Например: aziz-ar
3
Подтверди email
GitHub пришлёт письмо — нажми кнопку подтверждения в письме. Без этого страница не откроется.
2
Создать репозиторий
Это папка для файлов приложения · 2 минуты
1
Нажми «+» → «New repository»
Кнопка плюс в правом верхнем углу после входа.
+ → New repository
2
Заполни форму точно так:
Repository name: ar-crm
Visibility: Public ← обязательно!
Add README: ✓ включи
.gitignore: None
License: None
⚠️
Обязательно выбери Public — иначе бесплатный хостинг не работает. Данные приложения хранятся на твоём телефоне, не на GitHub, поэтому это безопасно.
3
Нажми зелёную кнопку «Create repository»
Откроется пустая страница репозитория.
Create repository
3
Загрузить файлы
3 файла — 2 минуты
1
На странице репозитория нажми «uploading an existing file»
Это ссылка в центре страницы — маленький текст под заголовком.
uploading an existing file
2
Перетащи все 4 файла в область загрузки
Или нажми «choose your files» и выбери все четыре сразу зажав Ctrl (на Windows) или ⌘ (на Mac).
📄
index.html
📄
manifest.json
📄
sw.js

3
Нажми зелёную кнопку «Commit changes»
Файлы загрузятся. Ничего не меняй в полях выше — просто нажми кнопку.
Commit changes
4
Включить GitHub Pages
Это делает файлы доступными по ссылке · 2 минуты
1
Нажми «Settings» вверху репозитория
Settings (вкладка в меню репозитория)
2
В левом меню найди «Pages»
Прокрути левую панель вниз — раздел «Pages» в группе «Code and automation».
Code and automation
  Pages ← нажми сюда
3
В разделе «Source» выбери «Deploy from a branch»
Если уже стоит — не меняй.
4
В разделе «Branch» выбери «main» → «/ (root)» → нажми Save
Branch: main   / (root)   Save
⏳
Подожди 1–2 минуты. Обнови страницу. Появится зелёная надпись с адресом сайта.

5
Установить на оба телефона
Android и iPhone — разные способы
📱
Азиз — Android (Chrome)
1. Открой ссылку в Chrome
2. Нажми три точки ⋮ вверху справа
3. Выбери «Добавить на главный экран» или «Установить приложение»
4. Нажми «Установить»
5. Иконка АР CRM появится на рабочем столе
🍎
Редван — iPhone (Safari)
1. Открой ссылку в Safari (не Chrome!)
2. Нажми кнопку «Поделиться» внизу — квадрат со стрелкой вверх
3. Прокрути список вниз → «На экран Домой»
4. Нажми «Добавить»
5. Иконка АР CRM появится на главном экране
⚠️
На iPhone работает только через Safari. Chrome на iPhone не поддерживает установку PWA.
6
Подключить Telegram уведомления
Через n8n · По желанию · 10 минут
🔗 Схема подключения
Когда любой партнёр создаёт объект или меняет статус — оба получают сообщение в Telegram.
1
Зайди на stiv122469.app.n8n.cloud → создай новый воркфлоу
2
Добавь узел Webhook — скопируй его URL (кнопка «Copy URL»)
3
Добавь узел Telegram → подключи токен бота АРВИС → в поле Text вставь: {{ $json.message }}
4
Продублируй Telegram узел — один с Chat ID Азиза, второй с Chat ID Редвана
5
Активируй воркфлоу — тумблер вверху справа
6
В приложении: вкладка ⚙️ Настройки → вставь URL вебхука → нажми Сохранить
💡
Chat ID найти просто: напиши боту @userinfobot в Telegram — он пришлёт твой ID числом. Попроси то же самое сделать Редвану.
